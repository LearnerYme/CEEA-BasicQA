# CEE Analysis - Basic QA

`Author`: Yige Huang
`Date`: 04.02.2026
`Version`: vallina.1.0

## Usage

1. Choose what histograms you need in `AnaMaker`;

2. Activate [CEE ROOT](https://gitee.com/CEESM/CeeRoot.git) environment, which dependes on [FAIR ROOT](https://github.com/FairRootGroup/FairRoot.git);

3. Compile with `make`;

4. Submit jobs onto SLURM system, reference [Manager](https://github.com/LearnerYme/CEEA-ManagerSystem.git).
